| feed response content → STIX pattern | IOC values from untrusted feeds are interpolated into STIX pattern strings |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-04-01 | Tampering | STIX pattern injection from URLhaus/Feodo IOC values | high | mitigate | Escape single quotes in all IOC values before string interpolation: value.replace("'", "\\'"). Applied in both URLhausFeed.normalize() and FeodoFeed.normalize(). |
| T-02-04-02 | Denial of Service | Oversized CSV response from URLhaus/Feodo | medium | mitigate | BaseFeed._fetch_with_retry() uses requests.get() with timeout=30; the caller in base.py retries on error. For very large responses, limit processing with list slicing if needed (URLhaus is typically ~5000 rows, Feodo ~150 rows). |
| T-02-04-03 | Tampering | CSV comment injection — '#' lines mistaken as data | medium | mitigate | Filter applied: [l for l in resp.text.splitlines() if not l.startswith("#")] eliminates all comment lines before DictReader. |
