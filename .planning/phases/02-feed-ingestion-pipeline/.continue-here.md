## Critical Advisory: docker compose profile

Always use **both** profiles for feed-orchestrator commands:
```bash
docker compose --profile platform --profile feeds <cmd>
```
`feed-orchestrator` has `depends_on: opencti` which is in `profiles: [platform]`. Using only `--profile feeds` gives `no such service: opencti`.
