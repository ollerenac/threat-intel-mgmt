  <action>
    Create services/intel-extractor/parser.py with two public functions:

    extract_pdf_text(pdf_bytes: bytes) -> str:
      Use `from PyPDF2 import PdfReader` (not `from pypdf import` — Pitfall 6).
      `import io` for BytesIO wrapper.
      reader = PdfReader(io.BytesIO(pdf_bytes))
      Extract text from each page: [page.extract_text() for page in reader.pages]
      Join non-None, non-empty results with "\n".
      If joined text is empty after .strip(), raise ValueError with message:
      "PDF appears to be image-based — no extractable text found"
      Return full_text.

    extract_url_text(url: str) -> str:
      URL scheme validation (SSRF guard): parse with urllib.parse.urlparse; if scheme not in
      ("http", "https"), raise ValueError with f"URL scheme '{scheme}' not allowed — only
      http and https are permitted".
      Primary: trafilatura.fetch_url(url) — check return for None (Pitfall 3: returns None,
      does not raise). If not None, call trafilatura.extract(downloaded); if result is not None
      and not empty, return result.
      Fallback: requests.get(url, timeout=15, headers={"User-Agent": "Mozilla/5.0"}),
      call raise_for_status(), parse with BeautifulSoup(resp.text, "html.parser"),
      return soup.get_text(separator="\n", strip=True).

    Module header docstring: "parser.py — Document text extraction for intel-extractor."
    Logging: logger = logging.getLogger(__name__); log at INFO when falling back to requests.
  </action>
