    Create services/intel-extractor/main.py:
      Imports: logging, uuid, FastAPI/BackgroundTasks/UploadFile/File/Form/HTTPException from fastapi,
      Optional from typing, extractor.jobs (the module-level dict) and extractor.run_extraction.
      Logging: basicConfig at module level (logging.basicConfig(level=logging.INFO));
      logger = logging.getLogger(__name__).
      app = FastAPI(title="intel-extractor", version="1.0.0")

      POST /extract endpoint (async def submit_extract):
        Parameters: background_tasks: BackgroundTasks, file: Optional[UploadFile] = File(None),
        url: Optional[str] = Form(None).
        If not file and not url: raise HTTPException(status_code=400, detail="provide file or url").
        Generate job_id = str(uuid.uuid4()).
        Initialize jobs[job_id] = {"status": "queued", "iocs_extracted": 0,
          "techniques_found": 0, "report_id": None, "error": None, "processing_time_s": None}
        BEFORE calling add_task (never after).
        If file: content = await file.read(); background_tasks.add_task(run_extraction,
          job_id, "pdf", content, None)
        Else: background_tasks.add_task(run_extraction, job_id, "url", None, url)
        Return {"job_id": job_id, "status": "queued"}

      GET /jobs/{job_id} endpoint (async def get_job):
        If job_id not in jobs: raise HTTPException(status_code=404, detail="job not found")
        Return {"job_id": job_id, **jobs[job_id]}

      GET /health endpoint (async def health):
        Return {"status": "ok"}

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → POST /extract | Untrusted file upload or URL string crosses API boundary here |
| FastAPI → BackgroundTasks | File bytes passed to background thread after response sent |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-03-05-01 | Denial of Service | UploadFile — unbounded file size | medium | mitigate | After `content = await file.read()`, check `len(content) > 50_000_000` (50MB) and return 413; log warning with file size |
| T-03-05-02 | Information Disclosure | GET /jobs/{id} exposes error field | low | accept | Error messages are internal service errors, not user secrets; demo scope; add auth in v2 |
| T-03-05-03 | Spoofing | job_id UUID sequential prediction | low | accept | uuid4() is random; not sequential — enumeration impractical |
| T-03-05-SC | Tampering | pip install in Docker build | high | mitigate | Package Legitimacy Audit in RESEARCH.md — all packages approved; slopcheck passed |
</threat_model>
