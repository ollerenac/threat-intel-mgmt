<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser (localhost:3000) → FastAPI services | Cross-origin fetch; browser enforces CORS preflight |
| FastAPI /feeds/status → Redis | Internal Docker network; Redis not exposed on host |
| FastAPI /stats → OpenCTI via pycti | Internal Docker network; pycti uses token auth |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-06-01-01 | Information Disclosure | CORSMiddleware on all 3 services | high | mitigate | allow_origins=["http://localhost:3000"] — explicit origin, never "*"; added in Task 1 api.py and Task 2 both files |
| T-06-01-02 | Tampering | XSS via IOC values in /feeds/status or /stats response | medium | mitigate | Responses are JSON (not HTML); React JSX renders JSON values as text nodes — never use dangerouslySetInnerHTML in frontend (enforced in Plan 03) |
| T-06-01-03 | Denial of Service | /stats blocks event loop on slow pycti call | medium | mitigate | asyncio.to_thread wraps _collect_threat_data — pycti I/O runs in thread pool, not on event loop |
| T-06-01-04 | Spoofing | Unauthenticated /feeds/status exposes feed run history | low | accept | Demo scope — no auth per REQUIREMENTS.md out-of-scope section; data is operational metadata, not IOC payloads |
</threat_model>
